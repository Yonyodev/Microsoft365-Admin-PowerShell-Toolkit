Project Overview

Purpose

This repository is a sanitized PowerShell portfolio project focused on Microsoft 365 administration, Exchange Online, Entra ID, Intune, reporting, and IT support automation workflows.

The goal is to demonstrate practical scripting organization, documentation habits, and secure handling of administrative automation examples.

Why This Project Exists

PowerShell is commonly used in IT support, systems administration, Microsoft 365 administration, and cybersecurity-adjacent workflows. This project is designed to organize PowerShell examples in a way that reflects real administrative tasks while avoiding the use of production data or company-specific information.

Planned Areas
Exchange Online mailbox and group administration
Microsoft 365 user and license management
Entra ID identity and group workflows
Intune endpoint management reporting
Audit-style reporting and exports
Reusable PowerShell templates
Secure scripting and sanitization practices

Portfolio Focus

This project is intended to show:

Ability to organize technical projects in GitHub
Understanding of Microsoft 365 administrative workflows
PowerShell scripting fundamentals
Security awareness when handling administrative data
Documentation habits useful in IT and cybersecurity roles

Data Handling

All examples in this repository use fictional users, domains, groups, devices, and outputs. No production data, tenant information, company-specific configurations, access tokens, credentials, or confidential information are included.
